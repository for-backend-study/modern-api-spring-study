## 📝 진행 방식

* 매주 일요일 밤 11시 59분까지, 해당 주의 실습 코드를 PR로 제출합니다.
* 각자 디렉토리를 만들어 그 안에 실습 내용을 정리합니다.
* PR은 자신의 브랜치에서 생성하며, 본인이 직접 Merge합니다.
* 실습 외에 새롭게 알게 된 내용이나 트러블슈팅 사례가 있을 경우, GitHub Issue에 정리합니다.
